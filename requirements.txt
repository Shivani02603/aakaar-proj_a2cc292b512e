**Primary Users:** Users who upload Excel files and ask natural-language questions about data  
**Tech Constraints:** Python FastAPI, Next.js, PostgreSQL with pgvector extension, Docker Compose, OpenAI text-embedding-3-small, Google Generative AI SDK (gemini-2.0-flash)

**Project Summary:** This project builds a web application that allows users to upload Excel (.xlsx) files and ask natural-language questions about the data. The system ingests Excel files by parsing them with pandas, chunking the content, embedding chunks with OpenAI, and storing vectors in pgvector. On user queries, it retrieves relevant chunks via cosine similarity and generates answers with source citations using Google's Gemini model, all within a session-managed chat interface.
